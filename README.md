## node-web-scraping

### Dependencies

- node.js
- npm

### How to install

<pre>
git clone https://github.com/clara-jr/node-web-scraping.git
cd node-web-scraping
npm install
</pre>

### How to configure

+ The server is configured to listen in port 4000. You can modify it.
+ Now you can see the app in port 4000: http://localhost:4000

### How to run

<pre>
node app.js
</pre>
